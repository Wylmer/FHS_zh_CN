## 要求

`/`文件夹下要求有如下文件夹或符号链接。

文件夹	|描述
--------|-------------------------
bin	|必需命令的二进制文件
boot	|引导加载程序的静态文件
dev	|设备文件
etc	|主机特定的系统配置
lib	|必需的共享库和内核模块
media	|可移动媒质的挂载点
mnt	|临时挂载一个文件系统用的挂载点
opt	|外部应用程序软件包
sbin	|必需的系统级二进制文件
srv	|本系统所提供服务的数据文件
tmp	|临时文件
usr	|二级层次结构
var	|变量数据

以上列出的每个文件夹的具体规范在下面各小节细述。`/usr`和`/var`各由完整的一节来叙述，因为它们比较复杂。
